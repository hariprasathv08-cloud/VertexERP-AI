# 🚀 VertexERP AI – AI Powered ERP Automation Platform

VertexERP AI is a full-stack enterprise automation platform that uses Artificial Intelligence to simplify invoice processing, vendor management, and business analytics.

## ✨ Features

- 🤖 AI-powered invoice extraction using Gemini AI
- 📄 PDF & image invoice processing
- 🏢 Vendor management system
- 📊 Real-time ERP analytics dashboard
- 📘 Live Excel ledger synchronization
- 🔐 JWT-based authentication & role management
- 🔔 Activity notifications and audit tracking
- 💬 AI ERP assistant for business insights

## 🛠 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript
- Chart.js

**Backend**
- Python
- FastAPI
- REST APIs

**Database**
- SQLite
- SQLAlchemy

**AI Integration**
- Google Gemini AI

**Security**
- JWT Authentication
- Role-Based Access Control
